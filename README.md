## クリップボード文字列正規化ツール

### 機能:
1. クリップボード内のテキストファイルを正規化します。
2. クリップボード内のテキストファイルを括弧(【】)で囲みます。

### 使用方法:

- 前準備(normalize.json):
  - "shortcut_key"に、処理に割り当てるショートカットキーを記載します
  - "translate"配下に、正規化(置換)するパターンを記載します。

- プログラム実行:
  - `python ClipBoardNormalizer.py`

- 正規化処理:
  - "shortcut_key"で定義したショートカットキーを押すと、クリップボード内のテキストが正規化されます
